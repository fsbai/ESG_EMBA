def age(born):
    
    if born is not None:
        born = str(born)
        born = datetime.strptime(born, "%d/%m/%Y %H:%M:%S").date()
        if ((born.year > annee_reference) or (born.year < 1900)):
            return 0
        today = datetime(annee_reference, 12, 31).date()
        return today.year - born.year - ((today.month, 
                                      today.day) < (born.month,

    else:
        return 0

def Scatter_Plot(my_dataframe, col, label_Y, label_X, Titre):
   
    # Trac du scatter plot
    n, bins, patches = plt.hist(my_dataframe[col], bins=10)
    pourcentages = [p*100 for p in n]
    for i in range(len(patches)):
        plt.text(bins[i], n[i], f'{pourcentages[i]:.1f}%', ha='center', va='bottom')

    plt.ylabel(label_Y)

  • Datereadhesion : 39% des clients n'ont pas de date de readhsion. C'est dire seul 62% de la clientle qui a renouvel leur adhsion. Nous allons dduire de cette colonne une variable catgorielle qui indique si le client est : nouveau, ancien avec radhesion, ancien sans radhesion.

Note :

  • Variables concernes : DATENAISSANCE, DATEREADHESION, CODEINSEE, PAYS
  • DATENAISSANCE : Pour traiter les valeurs manquantes de la variable , nous allons repartir les valeurs de cette variable en diffrentes tranche d'ge. Les valeurs manquantes seront dans la tranche d'age "sans age".
  • DATEREADHESION : Cration d'une variable catgorielle qui distingue les clients qui ont renouvells de ceux qui n'ont pas renouvells leur adhsion.
  • Codeinsee : Seul 3% du jeu de donnes manquant. ces 3% ne sont pas significatifs, nous supprimerons ces enregistrements.
  • PAYS : Nous garderons uniquement la france comme pays.
