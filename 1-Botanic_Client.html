On retrouve 3 groupes principaux qui sont leaders de Elles se repartissent par zone gégraphiques.

def age(born):
    
    if born is not None:
        born = str(born)
        born = datetime.strptime(born, "%d/%m/%Y %H:%M:%S").date()
        if ((born.year > annee_reference) or (born.year < 1900)):
            return 0
        today = datetime(annee_reference, 12, 31).date()
        return today.year - born.year - ((today.month, 
                                      today.day) < (born.month, 
                                                    born.day))
    else:
        return 0

def DureeAdhesion(row):
    DateDebut = row['DATEDEBUTADHESION']
    if DateDebut is not None:
        DateDebut = str(DateDebut)
        DateDebut = datetime.strptime(DateDebut, "%d/%m/%Y %H:%M:%S").date()
        if (DateDebut.year <= annee_reference):
            return (annee_reference - DateDebut.year)*12 + (12 - DateDebut.month)

            return 0

def Scatter_Plot(my_dataframe, col, label_Y, label_X, Titre):
   
    # Tracé du scatter plot
    n, bins, patches = plt.hist(my_dataframe[col], bins=10)
    pourcentages = [p*100 for p in n]
    for i in range(len(patches)):
        plt.text(bins[i], n[i], f'{pourcentages[i]:.1f}%', ha='center', va='bottom')
    plt.xlabel(label_X)
    plt.ylabel(label_Y)
    plt.title(Titre)
    plt.show()

    # Calcul des pourcentages de chaque modalité
    pourcentages = (mydataframe[col].value_counts(normalize=True)* 100).apply(lambda x: '{:.1f}%'.format(x))

  • Datenaissance : 39% des clients sans date naissance. Nous allons calculer l'age pour les dates de naissance valides, ensuite nous regrouperons les âges par tranche d'âges. Ainsi nous créerons une variable catégorielle avec des tranches d'âges. Ceux qui n'ont pas d'age renseigné seront définis dans une modalité à part.
  • Datereadhesion : 39% des clients n'ont pas de date de readhésion. C'est à dire seul 62% de la clientèle qui a renouvelé leur adhésion. Nous allons déduire de cette colonne une variable catégorielle qui indique si le client est : nouveau, ancien avec réadhesion, ancien sans réadhesion.

Note :

  • Variables concernées : DATENAISSANCE, DATEREADHESION, CODEINSEE, PAYS
  • DATENAISSANCE : Pour traiter les valeurs manquantes de la variable , nous allons repartir les valeurs de cette variable en différentes tranche d'âge. Les valeurs manquantes seront dans la tranche d'age "sans age".
  • DATEREADHESION : Création d'une variable catégorielle qui distingue les clients qui ont renouvellés de ceux qui n'ont pas renouvellés leur adhésion.
  • Codeinsee : Seul 3% du jeu de données manquant. ces 3% ne sont pas significatifs, nous supprimerons ces enregistrements.
  • PAYS : Nous garderons uniquement la france comme pays.
